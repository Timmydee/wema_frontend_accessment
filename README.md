## Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Timmydee/wema_frontend_accessment.git

   ```  
2. **Install dependencies**
   ```bash
   npm install
   ```

   ```bash
   npm run dev
   ```  
4. **Open in browser**
   Visit `http://localhost:5173` *(or your local development port)*
